# NotebookLM-Deutsch-Lernen
Primeiro projeto feito durante o bootcamp de Python para análise e automação de dados da Accenture, tem como objetivo utilizar a ferramenta da Google NotebookLM para fazer uma base de dados de verdade universal única, como forma de evitar os "delírios" de IAs generativas e o tema que eu escolhi foi a língua Alemã.
